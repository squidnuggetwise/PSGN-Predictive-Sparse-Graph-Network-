# PSGN-Predictive-Sparse-Graph-Network-
a neurosymbolic ai architecture made in c, made to recreate how the human brain learns algorithms using sparse distributed respresentations (SDR) and shift-XOR binding to learn and do math equations without normal boring neural nets

the kinds of ai that everyone is focusing on these days were bothering me, transformers have big problems like catastrophic forgetting and hallucinations, the PSGN solves all the problems that those architectures have by building a deterministic, sparse associative memory graph. it learns to rules of an algorithm and proceeds to autoregressively unroll the computation steps to solve maths problems it hasnt seen before in its training data
